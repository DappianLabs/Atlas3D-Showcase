## What is Atlas3D?
<img width="1910" height="994" alt="Opera Snapshot_2026-06-01_194516_www atlas3d space" src="https://github.com/user-attachments/assets/731dd1d5-183f-4f3b-ae93-569794af9656" />

Atlas3D is a platform for viewing and exploring interactive 3D models online.

Users can rotate models, zoom in on details, click individual parts to learn what they are, ask questions about the model to AI, and take quizzes generated from the model's content by AI.

Artists and educators can upload their own models, label important parts through a visual editor, and publish them for others to explore.

This repository contains information about the platform. The application source code is private.

### Interactive 3D Viewer

View models directly in the browser with support for rotation, zooming, and part selection.

### Part Identification

Click individual parts of a model to view names, descriptions, and related information.

### X-Ray Mode

Hide or isolate components to better understand internal structures.

### AI Assistant

Ask questions about the currently selected model and receive contextual answers.

### Quiz System

Generate quizzes from model data to help users learn and test their understanding.

### Creator Uploads

Upload GLB or GLTF models, label parts through the web interface, and submit them for publication.

### Global Delivery

Models are optimized and delivered through a CDN for fast loading worldwide.

### Cross-Platform Support

Works on desktop, tablet, and mobile devices.

## Who Uses Atlas3D?

### Students and Educators

Use interactive 3D models for teaching and learning.

### 3D Artists

Publish models in a format that visitors can explore instead of viewing static images.

### Researchers and Hobbyists

Study complex objects, systems, and structures in an interactive environment.

---

## How It Works

### For Visitors

1. Open a model from the gallery.
2. Rotate, zoom, and inspect individual parts.
3. Ask questions or take quizzes related to the model.

### For Creators

1. Upload a GLB or GLTF model.
2. Label important components using the built-in editor.
3. Submit the model for review and publication.
